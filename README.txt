 1. KPRとは

KPRは、Webサイトの構築・管理を補助するソフトです。

 1.1 特徴

 * WWW UI
 * 「ディレクトリ」作成機能
 * 「概要文」付与機能
 * ナビゲーションリンクの自動生成機能
 * 最終更新時刻の自動挿入機能
